# Curso Single Page Aplication usando Vanilla JS

## Install
```bash
$ git clone https://github.com/rafnixg/curso-spa-platzi.git
$ cd curso-spa-platzi
$ npm i
```

## scripts
```bash
$ npm run build
$ npm run start
```
